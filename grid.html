Les langages de programmations

Un langage de programmation est un ensemble de règles et de syntaxes qui permettent de communiquer avec un ordinateur pour lui donner des instructions precises. La formation ou cursus unniversitaire nous permet de developper des comptences en plusieurs langages de programmation tel que: le html,css,javascript et php pour la programmation web; le c , c++, haskell ,etc...

LANGAGES ET DESCRIPTIONS

HTML:

langage interpreté;utilisé en programmation WEB et permet de creer la struture de d'une page web.

CSS:

Pour Cascading StyleSheet est un langage utilisé en programmation WEB et permet la mise en forme d'une page web.

JAVASCRIPT:

Il permet de creer les interactions machine-utilisateur.

PHP:

Langage interpreté du coté serveur,il permet de traiter les données d'une page web

C:

Langage compilé qui permet de creer des applications de burau.

C++:

Fesant partir de la famille des langages compilés ,il est orienté objet et permet de creer des applications de burau.
